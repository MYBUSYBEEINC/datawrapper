# ChartDescription

Provides input text fields such as chart title, description, source etc. Commonly used in the annotate step of the chart editor.


```html
<ChartDescription />

<script>
    import { ChartDescription } from '@datawrapper/controls';
    // or import directly via
    // import ChartDescription from '@datawrapper/controls/editor/ChartDescription.html';
    
    export default {
        components: { ChartDescription }
    };
</script>
```
