# TypeAheadInput

A type-ahead input component. When you start typing,
different possible options will be presented in a dropdown.

```html
<TypeAheadInput
    id="unique-identifier"
/>

<script>
    import { TypeAheadInput } from '@datawrapper/controls';
    // or import directly via
    // import TextInput from '@datawrapper/controls/TypeAheadInput.svelte;

    let name = '';
</script>
```
